Movie Matcher:
A dynamic web application that helps users discover films by filtering through the TMDB database in real-time.

Key Features:
Dynamic Filtering: Processes user preferences to fetch and display relevant movie data instantly.

Real-time Database: Utilizes Firebase for seamless data management and user state.

Responsive UI: A clean, interactive frontend built with React.

Tech Stack:

Front End: React
Back End: Firebase 
Data: TMDB API
Stlying: Tailwind CSS
